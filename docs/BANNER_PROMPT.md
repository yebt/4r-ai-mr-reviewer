# Repo banner — image generation prompt

Aspect: **1280×640 (2:1)** — GitHub social preview.

## Prompt

> A minimalist technical banner for an open-source AI code-review tool named
> **"4R"**. Wide 2:1 composition on a near-black canvas (#0A0A0B) with a faint
> hairline grid. On the left: a bold monospace wordmark **"4R"** with a small
> acid-lime tick, and a stacked spec-sheet list of uppercase mono labels —
> **RISK · READABILITY · RELIABILITY · RESILIENCE** — separated by thin
> dividers. On the right: abstract line-art of a merge / pull-request flow —
> two branches converging into a single node — drawn as a clean blueprint
> wireframe in thin light strokes, with **one node highlighted in flame
> orange**. A slim **acid-lime** meter bar and a large percentage number as an
> accent detail. Aesthetic: borderless, industrial, editorial, Swiss-technical,
> data-terminal, spec-card. Flat vector, high contrast, generous negative
> space. Two accent colors only.

## Palette

| Role | Hex |
|---|---|
| Canvas | `#0A0A0B` |
| Ink (text/strokes) | `#EDEDF0` |
| Muted | `#7C7C85` |
| Hairlines | `#2A2A30` |
| Accent (lime) | `#D6FF3F` |
| Accent 2 (flame) | `#FF5A1F` |

## Avoid

Gradients, glossy 3D, drop shadows, photorealism, stock-photo people, robot
mascots, rainbow palettes, clutter. Keep it flat, technical, and mostly empty.

## Tagline options (optional overlay, mono uppercase)

- `4R QUALITY GATE — RISK · READABILITY · RELIABILITY · RESILIENCE`
- `AI CODE REVIEW FOR GITLAB & GITHUB MERGE REQUESTS`
